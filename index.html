1. Factory Constructor

Factory constructor adalah constructor khusus yang tidak selalu membuat instance baru. Ia bisa mengembalikan instance dari cache, subclass, atau melakukan logika kompleks sebelum membuat objek.


// ==========================================
// Nama  : ADRIEL NOFRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A
// ==========================================

      class MyClass {
  MyClass();
  factory MyClass.create() => MyClass();
}

void main() {
  var obj1 = MyClass();
  var obj2 = MyClass.create();
  print('Objek 1 dan 2 berhasil dibuat');
}

2. Singleton Pattern

Singleton memastikan sebuah kelas hanya memiliki satu instance selama program berjalan, serta menyediakan akses global ke instance tersebut.


// ==========================================
// Nama  : ADRIEL NOFRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A
// ==========================================
      class DatabaseConnection {
  DatabaseConnection._internal();
  static final DatabaseConnection _instance = DatabaseConnection._internal();
  factory DatabaseConnection() => _instance;
  void connect() => print('Connected to database');
}

void main() {
  var db1 = DatabaseConnection();
  var db2 = DatabaseConnection();
  print(identical(db1, db2)); // true
}

3. Object Pool Pattern

Object Pool menyimpan kumpulan objek siap pakai untuk digunakan kembali, mengoptimalkan performa dan menghemat resource.


// ==========================================
// Nama  : ADRIEL NOFRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A
// ==========================================
  class ConnectionPool {
  final String host;
  final int port;
  static final Map _cache = {};
  ConnectionPool._(this.host, this.port);
  factory ConnectionPool(String host, int port) {
    String key = '$host:$port';
    if (_cache.containsKey(key)) return _cache[key]!;
    var conn = ConnectionPool._(host, port);
    _cache[key] = conn;
    return conn;
  }
}

void main() {
  var conn1 = ConnectionPool('localhost', 5432);
  var conn2 = ConnectionPool('localhost', 5432);
  print(identical(conn1, conn2)); // true
}

4. Mengembalikan Subclass

Factory dapat mengembalikan instance dari subclass berbeda berdasarkan parameter yang diberikan.


 // ==========================================
// Nama  : ADRIEL NOFRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A
// ==========================================
      abstract class Shape {
  void draw();
  factory Shape(String type) {
    switch (type) {
      case 'circle': return Circle();
      case 'square': return Square();
      default: return Circle();
    }
  }
}

class Circle implements Shape {
  void draw() => print('Drawing circle');
}
class Square implements Shape {
  void draw() => print('Drawing square');
}

5. Factory untuk Configuration

Factory pattern untuk memuat konfigurasi dari berbagai environment (development, staging, production).


// ==========================================
// Nama  : ADRIEL NOFRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A
// ==========================================
      class AppConfig {
  final String environment;
  final String apiUrl;
  AppConfig._(this.environment, this.apiUrl);
  factory AppConfig.fromEnvironment(String env) {
    if (env == 'dev') return AppConfig._('development', 'http://localhost:3000');
    return AppConfig._('production', 'https://api.example.com');
  }
}

10. Immutable Objects + Cache

Objek tidak berubah (immutable) dengan cache untuk mendaur ulang instance yang sama.


// ==========================================
// Nama  : ADRIEL NOFRA MARCELINO
// NIM   : 251410017
// Kelas : SI2A
// ==========================================
      class Color {
  final int r, g, b;
  static final Map _cache = {};
  Color._(this.r, this.g, this.b);
  factory Color.rgb(int r, int g, int b) {
    String key = '$r,$g,$b';
    if (!_cache.containsKey(key)) _cache[key] = Color._(r, g, b);
    return _cache[key]!;
  }
}
