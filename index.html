1. Factory Constructor (Dasar)

Factory constructor tidak selalu membuat instance baru. Bisa mengembalikan dari cache, subclass, atau logika khusus. Seperti bintang jatuh yang bisa lahir dari debu kosmik yang sama.

// ========== ADRIEL NOFRA MARCELINO ==========
// NIM: 251410017
class MyClass {
  MyClass();
  factory MyClass.create() => MyClass();
}

void main() {
  var obj1 = MyClass();
  var obj2 = MyClass.create();
  print('Objek 1 & 2 dibuat dari pabrik bintang ✨');
}

2. Singleton Pattern

Memastikan hanya satu instance sepanjang siklus, seperti rasi bintang yang hanya satu di langit malam.

// ADRIEL NOFRA MARCELINO - 251410017
class DatabaseConnection {
  DatabaseConnection._internal();
  static final DatabaseConnection _instance = DatabaseConnection._internal();
  factory DatabaseConnection() => _instance;
  void connect() => print('Connected to cosmic DB');
}
void main() {
  var db1 = DatabaseConnection();
  var db2 = DatabaseConnection();
  print(identical(db1, db2)); // true
}

3. Object Pool (Cache Connection)

Mengelola koneksi reusable seperti meteor yang kembali ke sabuk asteroid untuk dipakai lagi.

// ADRIEL NOFRA MARCELINO - 251410017
class ConnectionPool {
  static final _cache = {};
  final String host; final int port;
  ConnectionPool._(this.host, this.port);
  factory ConnectionPool(String host, int port) {
    String key = '$host:$port';
    if (_cache.containsKey(key)) return _cache[key];
    var conn = ConnectionPool._(host, port);
    _cache[key] = conn;
    return conn;
  }
}
void main() {
  var a = ConnectionPool('localhost', 5432);
  var b = ConnectionPool('localhost', 5432);
  print(identical(a, b)); // true
}

4. Factory Mengembalikan Subclass

Factory bisa mengembalikan berbagai turunan sesuai parameter, seperti rasi bintang berbeda tergantung musim.

// ADRIEL NOFRA MARCELINO - 251410017
abstract class Shape { void draw(); }
class Circle implements Shape { void draw() => print('🌟 Lingkaran Bintang'); }
class Square implements Shape { void draw() => print('⬛ Kotak Meteor'); }
factory Shape(String type) {
  if(type=='circle') return Circle();
  return Square();
}

5. Factory Konfigurasi (Env)

Membuat konfigurasi berdasarkan environment — seperti menyesuaikan kecerahan bintang dengan latar langit.

// ADRIEL NOFRA MARCELINO - 251410017
class AppConfig {
  factory AppConfig.fromEnvironment(String env) {
    if(env == 'dev') return AppConfig._dev();
    return AppConfig._prod();
  }
  AppConfig._dev();
  AppConfig._prod();
}

6. Factory dengan Named Constructor

Multiple factory dengan nama deskriptif — seperti memberi nama pada setiap bintang terang.

// ADRIEL NOFRA MARCELINO - 251410017
class User {
  User._(this.name);
  final String name;
  factory User.guest() => User._('Guest Star');
  factory User.normal(String name) => User._(name);
}

7. Factory Generic (Response)

Pola factory yang type-safe untuk berbagai tipe data, seperti debu kosmik yang bisa membentuk berbagai objek langit.

// ADRIEL NOFRA MARCELINO - 251410017
abstract class Response<T> {
  factory Response.success(T data) = SuccessResponse<T>;
  factory Response.error(String err) = ErrorResponse;
}

8. Factory Metode Pembayaran

Memproduksi berbagai payment method — fleksibel seperti berbagai jalur cahaya bintang.

abstract class Payment { void pay(double amount); }
factory Payment(String type) {
  if(type == 'card') return CardPayment();

9. Immutable Objects dengan Cache

Objek warna yang tidak berubah dan di-cache, hemat memori layaknya konservasi energi di jagat raya.

// ADRIEL NOFRA MARCELINO - 251410017
class Color {
  static final _cache = {};
  final int r,g,b;
  Color._(this.r,this.g,this.b);
  factory Color.rgb(int r,int g,int b) => _cache.putIfAbsent('$r,$g,$b', () => Color._(r,g,b));
}

11. Latihan ShapeFactory

Bangun objek shape dengan mudah menggunakan factory — bentuk geometris antar bintang.

// ADRIEL NOFRA MARCELINO - 251410017
factory Shape(String type) {
  switch(type) {
    case 'star': return StarShape();
    default: return CircleShape();
  }
}

12. Factory vs Static Method

Perbedaan mendasar: factory constructor bisa digunakan dalam pewarisan, static method lebih ke utility.

// ADRIEL NOFRA MARCELINO - 251410017
class Example {
  factory Example.create() => Example._();
  static Example.createStatic() => Example._();
  Example._();
}
